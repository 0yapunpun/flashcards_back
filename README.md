## Setup
To run this project: 

1. Clone the repo
   ```sh
   git clone https://github.com/0yapunpun/flashcards_back.git
   ```
2. Import config file 
 
3. Install NPM packages
   ```sh
   npm install
   ```
4. Init the project
   ```sh
   npm start
   ```
   
## Built with 
- NodeJS
- Express
- MongoDB
